# 🚗 Rent App

A modern Flutter car rental application that allows users to browse, view details, and explore cars with a clean UI and smooth experience.

---

## 📱 Features

- Browse available cars
- View car details
- Google Maps integration
- Clean and modern UI
- Onboarding screens
